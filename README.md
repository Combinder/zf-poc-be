### How to Run locally:
clone down this repository: 
> ```git clone git@github.com:Combinder/zf-poc-be.git```

cd into the repository:
> ```cd zf-poc-be```
>
Run create a `Python3` virtual env and activate it: 
> ```python3 -m venv .venv```
> 
> ```source .venv/bin.activate```

Create a .env file at the root folder
`touch .env` and credential will be shared separately (add them to `.env`)

Install project requirements:

```pip install -r requirements.txt```

run the project:
```
python app.py
```
